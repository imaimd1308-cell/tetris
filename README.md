# 테트리스 게임

HTML, CSS, JavaScript로 만든 반응형 테트리스 웹게임입니다.

## 게임 실행

아래 주소에서 게임을 실행할 수 있습니다.

https://본인깃허브아이디.github.io/tetris/

## 조작 방법

### PC

* 왼쪽·오른쪽 방향키: 블록 이동
* 아래 방향키: 빠르게 내리기
* 위 방향키: 블록 회전
* 스페이스바: 블록 바로 내리기
* P키: 일시정지

### 스마트폰

화면 아래의 이동·회전·바로 내리기 버튼을 눌러 조작합니다.

## 사용 기술

* HTML
* CSS
* JavaScript
* GitHub Pages
